<h1 align="center">Introdução ao HTML?</h1>
<br>


| Perguntas-chave | Notas principais |
|---|---|
| O que significa HTML? | HyperText Markup Language |
| O que é Hypertext? | Textos com links que conectam documentos HTML entre si |
| O que é Markup Language? | Linguagem que usa tags para indicar ao navegador como exibir o conteúdo |
| O que são tags HTML? | Marcações que estruturam o conteúdo da página (ex: `h1` para indicar título) |
| Posso criar um site só com HTML? | Sim! É o único dos três arquivos (HTML, CSS, JavaScript) que funciona sozinho |
| Quais são as tags de heading? | `h1` até `h6` — do mais importante ao menos importante |
| Quem criou o primeiro site? | Sir Tim Berners-Lee, também inventor da Internet |

---

## Resumo 

> HTML é a espinha dorsal de qualquer site. Ele usa tags para marcar o conteúdo e dizer ao navegador como exibi-lo. A base da web são os hyperlinks: textos que conectam documentos HTML entre si, exatamente como no primeiro site criado por Tim Berners-Lee.
